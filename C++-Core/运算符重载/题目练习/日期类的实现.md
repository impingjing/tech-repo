# 日期类的实现

## 问题再现

> 
> 
> 
> 使用重载运算符（++，+=，<<等）实现日期类的操作。功能包括：
> 
> 1）设置日期，如果日期设置不符合实际，则设置为默认日期（1900年1月1日）
> 
> 2）在日期对象中向日期添加1或者加若干天（加入日期值后根据实际的日期进行月份、年份的变化）
> 
> 3）重载流插入运算符进行日期的输出，其中月份要用名称表示
> 

```cpp
#include <string>
#include <iostream>
using namespace std;

/* 请在这里填写答案 */

int main()
{
    int m,d,y;
    MyDate d1,d2,d3;
    cin>>m>>d>>y;
    d1.setDate(m,d,y);
    
    cin>>m>>d>>y;
    d2.setDate(m,d,y);
    
    cin>>m>>d>>y;
    d3.setDate(m,d,y);
    
    cout << "d1 is " << d1 << "\nd2 is " << d2;
    cout << "\n\nd1 += 7 is " << ( d1 += 7 );
    cout << "\n\n d2 is " << d2;
    cout << "\n++d2 is " << ++d2;
    cout << "\n\nTesting the prefix increment operator:\n"<< " d3 is " << d3 << endl;
    cout << "++d3 is " << ++d3 << endl;
    cout << " d3 is " << d3;
    cout << "\n\nTesting the postfix increment operator:\n"<< " d3 is " << d3 << endl;
    cout << "d3++ is " << d3++ << endl;
    cout << " d3 is " << d3 <<endl;
} 
```

## 程序实现

```cpp
// 判断是否是闰年
bool isLeapYear(int y)
{
    return (y%4==0 && y%100!=0) || (y%400==0);
}
//获取某年某月的天数
int getDay(int y,int m)
{
    int days[]={0,31,28,31,30,31,30,31,31,30,31,30,31};
    if(isLeapYear(y) && m==2)
        return 29;
    return days[m];
}
//
class Date{
    friend ostream& operator<<(ostream& cout,const Date& date);
private:
    int month;
    int day;
    int year;
public:
    // 默认构造函数
    Date()=default;
    // 有参构造函数
    Date(int m,int d,int y)
    {
        setDate(m,d,y);
    }
    // 设置日期
    void setDate(int m,int d,int y)
    {
        if(m>=1 && m<=12 && d>=1 && d<=getDay(y,m))
        {
            month=m;
            day=d;
            year=y;
        }
        else
        {
            month=1;
            day=1;
            year=1900;
        }
    }
    //重载+=
    Date& operator+=(int extra)
    {
        day += extra;
        while(day>getDay(year,month))
        {
            day -= getDay(year,month);
            month++;
            if(month>12)
            {
                year++;
                month=1;
            }
        }
        return *this;
    }
    //重载前置++
    Date& operator++()
    {
        return (*this)+=1;
     }
     //重载后置++
     Date operator++(int)
     {
         Date temp=*this;
         ++(*this);
         return temp;
     }
};
//重载流输出符
// 使用全局常量数组存储月份名称，避免在全局作用域写赋值语句
const string monthNames[] = {
    "", "January", "February", "March", "April", "May", "June", 
    "July", "August", "September", "October", "November", "December"
};
ostream& operator<<(ostream& cout,const Date& date)
{
    cout << monthNames[date.month] << " ";
    cout << date.day << ", ";
    cout << date.year;
    return cout;
}
```

## 疑难注意

### 1. 全局作用域下不能执行赋值语句

```cpp
//企图建立全局数组：
vector<string> arr(12);
arr[0]="January"; 
arr[1]="February";
// ...
```

在C++中，全局作用域只能进行变量的**声明**和**初始化**，不能执行像 `arr[0]="January";` 这样的赋值或函数调用语句。这会导致编译器报错。

正确做法：通过**静态的字符串数组**

```cpp
const string monthNames[] = {
    "", "January", "February", "March", "April", "May", "June", 
    "July", "August", "September", "October", "November", "December"
};
```

### 2. 逻辑BUG：+= 运算符跨年时的闰年判断失效

```cpp
if(isLeapYear(year)) {
    while(day>l_d[month]) { ... year++; ... }
} else {
    while(day>d[month]) { ... year++; ... }
}
```

在 `operator+=` 中，你只在 `while` 循环**外部**判断了一次 `isLeapYear(year)`

假设你要加 1000 天，year 会在 while 循环内部不断增加。但是，代码使用的是加天数之前那个年份的闰年状态。如果原来不是闰年，加了几年后变成了闰年，程序依然会按照平年处理（即一直使用 d[month] 而不是 l_d[month]），导致日期计算完全错误。

正确做法：
我们需要将“获取当前月份天数”的逻辑提取成一个函数，并在 `while` 循环**内部**每次动态获取，这样无论年份怎么变，都能拿到正确的天数。

### 3. 代码结构优化

- 既然你已经写了 `+=` 运算符，那么 **前置 `++`** 其实就是 `*this += 1;`
- 而 **后置 `++`** 就是保存当前状态后，调用前置 `++`。
```cpp
// 注意：用前置++实现后置++时，不要用 (*this)++; !
MyDate operator++(int){
    MyDate temp=*this;
    ++(*this); //✅️
    // (*this)++; //❌️
    return temp;
}
```

### 4. 重载输出流时，加入const保护

```cpp
// 重载 << 运算符
ostream& operator<<(ostream& out, **const MyDate& date**) {
    out << monthNames[date.month] << " " << date.day << ", " << date.year;
    return out;
}
```
